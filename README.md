# 🎴 Demon System

> **Интерактивный чат-бот на базе Kivy/KivyMD с двумя AI демонами**

---

## 🚀 Описание

**Demon System** — это мобильное приложение для Android, реализованное на **Python** с использованием фреймворка **Kivy** и компоненты **KivyMD**.

Приложение предоставляет интерактивный интерфейс для общения с двумя виртуальными "демонами":
- 🤖 **AI Demon** — искусственный интеллект
- 💎 **Gem Demon** — второй виртуальный ассистент

*В настоящий момент чаты работают в тестовом режиме с имитацией ответов.*

---

## 📱 Функции

✅ **Две независимые диалоговые системы**  
✅ **Интуитивный мобильный интерфейс** (Portrait режим)  
✅ **Боковое меню навигации** (Navigation Drawer)  
✅ **Темная тема оформления** (Dark mode)  
✅ **Динамическое управление экранами**  
✅ **История сообщений** (в пределах сессии)

---

## 🛠️ Технологический стек

- **Python 3.x** — основной язык
- **Kivy** — фреймворк для мобильного интерфейса
- **KivyMD** — Material Design компоненты
- **Buildozer** — компиляция под Android (опционально)
